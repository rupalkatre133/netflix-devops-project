# Netflix DevOps Project

This is a Production-Level DevOps Project using:

- Jenkins
- Docker
- Terraform
- AWS EKS
- Kubernetes
- Helm
- ArgoCD

## Features

- CI/CD Pipeline
- Infrastructure Automation
- Kubernetes Deployment
- GitOps Workflow
- Docker Containerization

## Technologies

- AWS
- Terraform
- Docker
- Jenkins
- Kubernetes
- Helm
- ArgoCD

## Steps

1. Build Docker Image
2. Push Docker Image
3. Create AWS Infrastructure
4. Deploy EKS Cluster
5. Deploy Application
6. Automate using Jenkins
7. GitOps using ArgoCD
